## 一、背景与目标

### 1.1 背景
第一期报名页（index-v2.html）以「记路家计划」为主题，面向懂运动的内容创作者，核心卖点是运动体验 + 内容产出 + 官方身份认证。第二期主题切换为「梦回民国」，以莫干山深厚的民国历史底蕴为载体，打造一场山野实景剧本杀。

### 1.2 产品目标
- 保留一期已验证的页面结构与转化路径（Hero 冲击 → 权益陈列 → 价值锚定 → 倒计时 CTA）
- 全面替换内容层，从「运动身份招募」叙事转为「剧情角色招募」叙事
- 视觉氛围从「户外运动杂志风」转为「民国复古风」，强化沉浸感

### 1.3 叙事逻辑的关键转变

| 维度 | 一期（记路家） | 二期（梦回民国） |
|---|---|---|
| 用户角色 | 运动内容创作者 / 记路家 | 剧中角色 / 入局者 / 民国来客 |
| 核心动词 | 丈量、寻迹、记录 | 入戏、解谜、穿越、揭幕 |
| 核心资产 | 路线、视频、认证身份 | 剧本、角色、线索、结局 |
| 情绪钩子 | 官方认证 + 内容冷启动 | 悬念感 + 历史沉浸 + 圈层社交 |
| 时间主线 | 48小时倒计时 | 可沿用倒计时，语义改为「距开演/入局」 |

## 三、视觉风格更新建议

### 3.1 配色（CSS `:root` 变量）
| 变量 | 现状 | 建议 |
|---|---|---|
| --accent | #D97B2A（活力橙） | 民国色系候选：暗金 #B8860B / 朱砂红 #A63A2B / 墨绿 #2F4A3C，建议以暗金或朱砂为主强调色 |
| --bg | #F7F5F0 | 可加深做旧感，如宣纸米黄 #F3EDE0 |
| --dark | #0A0A0A | 可保留，或调为墨蓝 #14181F 增强年代感 |

### 3.2 字体与排印
- 中文主字体从 Inter/Noto Sans 的现代无衬线感，向 Noto Serif SC（宋体）倾斜，标题可加大 Ma Shan Zheng（书法体）使用比例
- 可选装饰元素：印章样式徽标、竖排文字点缀、繁体字标题点缀（正文保持简体，保证可读性）
- 英文花体 Lobster 建议移除或替换

### 3.3 图像处理
- 统一做旧滤镜方向：低饱和、暖褐色调、轻微暗角，模拟老照片质感
- 可在关键图片上叠加「民国报纸/邮戳/火漆」类装饰元素（CSS 或素材层实现）
